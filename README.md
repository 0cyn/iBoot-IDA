# ibootloader

this project was heavily inspired by https://github.com/argp/iBoot64helper

WIP Loader for Apple's SecureROM/iBoot

### Disassembler Support:

| Disassembler | Supported |
|--------------|-----------|
| IDA 7.0-7.5  | ✓         |
| Ghidra       | ✗         |
| Hopper       | ✗         |

# Filetype Support 

| File type                | Supported |
|--------------------------|-----------|
| arm32/64 SecureROM       | ✓         |
| arm32/64 iBoot/iBEC/iBSS | ✓         |
| SEPROM                   | ✗        |
